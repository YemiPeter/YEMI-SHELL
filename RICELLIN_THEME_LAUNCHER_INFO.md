# Ricelin Theme and Launcher Implementation

## Theme System Location

The Theme.qml file in Ricelin is located at:
```
.Ricelin/configs/quickshell/pill/Singletons/Theme.qml
```

## Theme System Structure

The theme system in Ricelin consists of:

1. **Theme.qml** - The main singleton that defines color properties and manages theming
   - Uses `pragma Singleton` to ensure single instance across the application
   - Contains both static and dynamic theming capabilities
   - Properties switch between static hex colors and dynamic colors from the `Dyn` singleton based on the `dyn` flag

2. **Dyn.qml** - The dynamic palette singleton that handles wallpaper-derived colors
   - Monitors a JSON file (`~/.cache/ricelin/colors.json`) generated by matugen
   - Updates theme properties in real-time when wallpaper changes
   - Provides Material You-style color extraction from wallpapers

3. **Walls.qml** - The wallpaper management singleton
   - Handles wallpaper selection, caching, and application
   - Coordinates with external scripts for wallpaper transitions
   - Manages wallpaper state and thumbnails

## Launcher Overlay Location

The launcher overlay in Ricelin is located at:
```
.Ricelin/configs/quickshell/launcher/
```

## Launcher Structure

The launcher system consists of:

1. **shell.qml** - The main launcher shell that creates the overlay window
   - Uses `ShellRoot` to create an overlay window
   - Implements IPC handlers for showing/hiding the launcher
   - Manages app usage statistics and fuzzy search

2. **Launcher.qml** - The main launcher UI component
   - Provides the visual interface for app launching
   - Contains search field, app list, and selection handling
   - Uses AppRow delegates for individual app entries

3. **AppRow.qml** - Individual app entry display component
   - Shows app icons, names, and descriptions
   - Handles selection and activation events

4. **lib/fuzzy.js** - JavaScript library for fuzzy search functionality

## Key Features

### Theming
- Dynamic theming based on wallpaper using matugen
- Static theming fallback with curated color palette
- Real-time theme updates when wallpaper changes
- Material You-inspired color extraction

### Launcher
- Overlay window positioned on top of other windows
- Fuzzy search for applications
- Usage-based app ordering
- Keyboard navigation support
- Transparent background with custom styling

## Integration Points

The theme system is integrated throughout the Ricelin shell:
- Colors are accessible as singleton properties across all components
- Dynamic theming is controlled by flags in the `Flags` singleton
- Palette updates happen automatically when wallpaper changes

The launcher integrates with:
- IPC system for show/hide commands
- Desktop entries for application discovery
- File system for usage statistics persistence
