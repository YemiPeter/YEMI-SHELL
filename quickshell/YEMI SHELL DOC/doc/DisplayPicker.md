# DisplayPicker

## 1. Component Overview

DisplayPicker is a reusable pill component that provides a display picker for the Yemi QuickShell desktop. It allows selecting and configuring monitors in the display settings surface.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/DisplayPicker.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Display.qml
- **Depends on**: DisplayLabel, display configuration

## 3. Component Hierarchy and Role

DisplayPicker is a reusable UI component that renders a display picker. It provides monitor selection and configuration controls in the display settings surface.
