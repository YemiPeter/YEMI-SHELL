# HFader

## 1. Component Overview

HFader is a reusable pill component that provides a horizontal fader for the Yemi QuickShell desktop. It is used in the mixer surface for audio volume control.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/HFader.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Mixer.qml
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

HFader is a reusable UI component that renders a horizontal fader. It provides audio volume control in the mixer surface.
