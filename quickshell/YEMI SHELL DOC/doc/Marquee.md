# Marquee

## 1. Component Overview

Marquee is a reusable pill component that provides scrolling text display for the Yemi QuickShell desktop. It is used to display long text that scrolls horizontally, such as track names or notification messages.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/Marquee.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Various pill surfaces that need scrolling text
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

Marquee is a reusable UI component that renders scrolling text. It provides a way to display long text within constrained spaces.
