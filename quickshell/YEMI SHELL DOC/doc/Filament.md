# Filament

## 1. Component Overview

Filament is a reusable pill component that provides a filament/material picker for the Yemi QuickShell desktop. It allows selecting visual materials or filaments for customization.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/Filament.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Look.qml or other appearance surfaces
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

Filament is a reusable UI component that renders a material/filament picker. It provides visual material selection controls.
