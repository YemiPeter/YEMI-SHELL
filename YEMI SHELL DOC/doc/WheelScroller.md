# WheelScroller

## 1. Component Overview

WheelScroller is a reusable pill component that provides mouse wheel scrolling behavior for the Yemi QuickShell desktop. It is used to enable scroll interactions on pill surfaces.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/WheelScroller.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Pill.qml and other scrollable surfaces
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

WheelScroller is a reusable UI component that enables mouse wheel scrolling. It provides scroll interaction for pill surfaces and other scrollable content.
