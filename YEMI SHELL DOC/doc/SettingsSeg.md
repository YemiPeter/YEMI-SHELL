# SettingsSeg

## 1. Component Overview

SettingsSeg is a reusable pill component that provides a segmented control for settings in the Yemi QuickShell desktop. It allows selecting between multiple options in a settings surface.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/SettingsSeg.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Settings and other settings surfaces
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

SettingsSeg is a reusable UI component that renders a segmented control. It provides a way to select between multiple options in settings surfaces.
