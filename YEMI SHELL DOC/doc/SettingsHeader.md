# SettingsHeader

## 1. Component Overview

SettingsHeader is a reusable pill component that provides a header for settings surfaces in the Yemi QuickShell desktop. It displays a title and optional back button for navigation within settings surfaces.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/SettingsHeader.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Settings and other settings surfaces
- **Depends on**: No external dependencies

## 3. Component Hierarchy and Role

SettingsHeader is a reusable UI component that renders a settings surface header. It provides consistent title display and navigation for settings surfaces.

SettingsHeader does not expose documented public properties in the source.

## 5. Signals

SettingsHeader does not define custom signals.

## 6. Methods

SettingsHeader does not define documented custom methods in the source.

## 7. Inter-Component Interactions

- **Settings.qml**: Uses SettingsHeader for surface header
- **Other settings surfaces**: Use SettingsHeader for consistent header styling

## 8. Usage Example

SettingsHeader is used as a child component within settings surfaces.
