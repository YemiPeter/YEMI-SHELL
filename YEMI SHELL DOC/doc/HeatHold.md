# HeatHold

## 1. Component Overview

HeatHold is a reusable pill component that provides a heat/hold indicator for the Yemi QuickShell desktop. It is used in the mixer surface to show audio fader hold states.

## 2. Project Structure and Dependencies

- **File**: `modules/pill/HeatHold.qml`
- **Imports**: `QtQuick`, `Quickshell`
- **Instantiated by**: Mixer.qml
- **Depends on**: No external dependencies
